Hello, ITDepartment!

Dit is een styleguide met (Bootstrap) HTML-elementen die reageren op de CSS van de applicatie (itd(-ie8).css). Zo kun je bijv. variabelen aanpassen in de SCSS-files en hier zien welke uitwerking dat heeft voor algemene Bootstrap HTML-structuren.

Let op: Het Bootstrap 3.0 grid-systeem wordt niet gebruikt in deze file. We maken gebruik van een eigen ITD-gridsysteem (css/itd-lib/_grid-itd.scss).
